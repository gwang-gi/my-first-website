태그 종류 공부 하기

  1. 글자 굵게 하기: strong 태그
  2. 밑줄 치기: u 태그
  3. 본문: h1 ~ h6 (숫자 클수록 작음) (줄바꿈도 됨)
  4. 줄바꿈: br 태그 (엔터키라고 생각하면됨) (닫는 태그 없음)
  5. 줄바꿈 및 단락 태그: p태그 --> br과 기능적으로는 같지만, 좀더 시작적으로 보여줌
  6. 꼭 암기 할 필요없이 검색엔진을 통해 검색해서 작업하면 된다!
  7. 태그를 하는 중요한 이유는 검색엔진, 시작장애인 등을 고려했을때 중요하다!

태그의 속성

  1. 태그만으로 부족할떄 쓰는 태그 (속성을 부여해야만 할때, 예를 들어 이미지 태그)
  2. 이미지 태그 unsplash.com: img라는 태그와 속성(src, width 등) (같은 폴더안에 이미지 저장후)

다른 것들

  1. (해드부분)title: 웹브라우저위에 나타나는 홈페이지 이름
  2. (해드부분)meta charset="utf-8": 비주얼스튜디오는 utf-8이기 때문에 한국어 쓸때 uft-8로 쓴다고 코딩을 해야함
  3. 본문은 body로, body를 설명하는 태그는 head태그, 이 둘을 감싸는건 html태그

링크 태크 a

a로 감싸고 href=""안에 링크 주소를 쓴다
